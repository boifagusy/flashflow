## Usage

1. Start the mobile development server on your computer:
   ```bash
   flashflow mobile
   ```

2. Access the mobile IDE from your smartphone or tablet by navigating to:
   ```
   http://[your-computer-ip]:8080
   ```

3. Start coding your FlashFlow applications directly from your mobile device!

## Requirements

- A computer running FlashFlow (Windows, macOS, or Linux)
- An Android or iOS device with a modern web browser
- Both devices must be on the same local network

## Benefits

- **No Laptop Required**: Code anywhere with just your smartphone or tablet
- **Simplified Workflow**: Streamlined development process optimized for touch interfaces
- **True Mobility**: Develop applications while on the go
- **Familiar Interface**: Intuitive touch-based controls designed for mobile devices
- **Python Flet Only**: Simplified architecture using only Python Flet for UI, removing HTML/CSS/JavaScript complexity

## Architecture

The mobile IDE is built entirely with Python Flet, providing a consistent experience across all platforms. This eliminates the need for HTML/CSS/JavaScript and simplifies the development stack to pure Python.
