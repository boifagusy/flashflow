# Assets Directory

This directory contains static assets used in the FlashFlow application.

## Structure

```
assets/
├── images/     # Image files (PNG, JPG, SVG, etc.)
├── icons/      # Icon files
└── fonts/      # Custom font files
```

## Usage

Place all static assets in their respective directories:

- **images/**: Place all image files here (logos, backgrounds, illustrations, etc.)
- **icons/**: Place icon files here (SVG icons, favicon, etc.)
- **fonts/**: Place custom font files here (WOFF, WOFF2, TTF, etc.)

## Best Practices

1. Use descriptive filenames (e.g., `logo-primary.svg` instead of `img1.svg`)
2. Optimize images for the web to reduce file sizes
3. Use vector formats (SVG) for icons when possible
4. Organize assets in subdirectories for larger projects
5. Keep a consistent naming convention
